# Ai-Powered-resume-Screener-for-fast-and-Accurate-Hiring
The AI Resume Screener is a Streamlit-based application designed to automate and enhance the resume analysis process using Google’s Gemini generative AI. It helps recruiters, HR professionals, and job seekers by evaluating resumes against specific job descriptions and generating intelligent insights such as match percentage, skill suggestions, job recommendations, and resume summaries.
The system begins by allowing users to upload resumes in PDF format. Using the PyPDF2 library, the application extracts text content from the resumes. This raw text is then cleaned and preprocessed using regular expressions to remove unwanted characters and whitespace. If applicable, users also input a job description into the system.
Once the resume and job description are ready, a custom prompt is constructed. This prompt combines the job description, resume text, and a specific query, such as "How well does this resume match the job description?" or "What are the missing skills?" The prompt is then sent to the Gemini 1.5 Pro model via the google.generativeai API, which generates a contextual response.
The model’s response is parsed to extract relevant information like percentage match scores, explanations, or skill recommendations. These results are then displayed interactively in the Streamlit interface using progress bars, tables, and expandable sections. In cases where multiple resumes are uploaded, the tool automatically ranks them based on their fit with the job description.
This AI-powered screener can be effectively used in recruitment processes, campus placements, and job recommendation platforms. It saves time, reduces manual screening effort, and ensures more accurate candidate-job alignment by leveraging the power of large language models.
